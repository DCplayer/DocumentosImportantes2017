Noticia Semanal
link: http://thehackernews.com/2017/02/fileless-malware-bank.html

Resumen:La noticia trata de un nuevo malware, el cual no necesitaba de alojamiento
de archivos en el disco que lo pudieran delatar o con los cuales pudieramos
buscar un trazo del proveedor del malware. Este tipo de ataque se aloja en la 
memoria Ram para que al momento de hacer reboot a la computadora, se borre cualquier
tipo de trazo del malware. Un malware enfocado para ser usado para ATM y extraccion
de dinero. 
Encontrado por un grupo de se3guridad de un banco, se descubrio una copia de 
Meterpreter, el cual utilizaba el PowerShell de Windows para directamente escribir
el codigo en la memoria o bien utilizaban Microsoft's NETSH para hacer un tunnel
proxy que se comunicaba con el C&C server.
